고객사에서 이미지 업로드 구현할 페이지입니다.
본 샘플은 이노디터에게 전달되어야 하는 내용만 표현하고자 단순화하여 제공한 샘플입니다.

이노디터는 본 샘플처럼 이미지 관련 내용만 전달받으면 되기 때문에
자체적으로 화면 페이지를 자유롭게 구성하신 후 전달값만 정확히 처리해 주시면 됩니다.
(전달값 - 본 샘플 페이지의 소스 부분을 확인 또는 매뉴얼 참조)




이노디터에서 제공하는 샘플을 사용하고자 하시는 경우
innoditor > js > customize.js 에서

var g_strUploadImageURL = "/innoditor/pop_simple_img.html";
-->
var g_strUploadImageURL = "/innoditor/pop_upload_img.html";

변경하신 후 매뉴얼과 샘플페이지를 참조하시어 고객사 업로드 관련 서버 모듈과 연동하셔야 합니다.
(이노디터 제공 샘플 페이지 자체가 업로드를 진행하지는 않습니다)


이미지전달(기본) -> 이노디터 체험게시판 기준
ex) 이미지 옆으로 이미지 높이만큼 멀티라인 글을 작성할 수 있습니다.
이미지전달(확장) -> 정렬 등의 사용자 정의 추가 설정
ex) 정렬을 설정하지 않는 경우 이미지 바로 아래 글을 작성할 수 있습니다.
이미지전달(확장)은 pop_upload_img.html 상에 주석으로 제공되어 있으며,
사용하시고자 하는 경우 기본설정으로 되어 있는 부분을 주석처리 하시고 확장부분을 주석해제하여
사용하시면 됩니다.

이미지 전달방식을 사용하세요
