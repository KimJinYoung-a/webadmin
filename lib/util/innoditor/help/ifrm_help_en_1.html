Create Table
If you press Create table Dropdown button, selecting row X column will appear.
Then place the mouse on rows and columns, and set your own table.
Insert Row
If you highlight single row, cell or some rows, inserting row button will activated.
Pressing the Insert row button will insert the number of row as much as before the highlighted row.
Delete Row
If you highlight some or single row, deleting row button will activated.
Pressing the delete row button will delete highlighted area.
Insert Col
If you highlight single column, cell or some columns, inserting column button will activated.Pressing the Insert
column button will insert the insert the number of columns as much as before the highlighted columns.
Delete Col
If you highlight some or single column, deleting column button will be activated.
Pressing the delete column button will delete highlighted area.
Merge Cell
If you highlight number of cells, merging cell button will be activated.
Pressing the merge cell button will merge highlighted area.
Split Row
If split cell is on normal state, then split cell button will be activated in highlighted table.
Pressing split cell button will split highlighted area.
Split Col
If split column is on normal state, then split column button will be activated in highlighted table.
Pressing split column button will split highlighted area.
Table Templete
Select the table template you want to create. It will format on each style of table.

Cell Property(2) - Width / Height / Align
In order to change the cell properties, highlight the part of cells, and press Cell Property.
Then, Popup window will appear. On the right top, pressing size/array button, you can set size and layout.
Select cell This shows cell property before
Cell properties pop up window.
If you click the selected cell,
it will be selected or unselected
and the other cells can be selected
in same way.
Width/Height Set the selected cells width/height.
If it is not settled, you may not set
cell size. However, you may
set width or height separately.

vertical alignment.
(Horizontal - left, center, right
