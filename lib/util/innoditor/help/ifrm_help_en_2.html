Cut Copy Paste Delete Select All
Cut
Firstly highlight the fragment you want to cut.
Go to menu, Edit -> Cut, pressing the combination of Ctrl+X on key board or using popup window menu.
Copy
Highlight the fragment you want to copy.
Go to menu, Edit -> Copy, pressing the combination of Ctrl+C on key board or using popup window menu.
Paste
Highlight the fragment you want to paste.
Go to menu, Edit -> Paste, pressing the combination of Ctrl+V on key board or using popup window menu.
Delete
Highlight the fragment you want to delete.
Go to menu, Edit -> Delete, pressing the Delete on the keyboard or using popup window menu.
Select All
To highlight all your text, Go to menu, Edit -> Select All, pressing the Ctrl+A on the keyboard
