Create Link
Highlight the area you want to Hyper Link, and press Connect Hyperlink button.
Set the target of your link when hyperlink properties pop up window appears.
=> URL (essential input) : specifies the address you want your link to refer to.
=> Target (optional input) : Parent window, New window, Top-most window, Current window
=> Tooltip (optional input) : If you move cursor on connected link, tooltip (balloon) will appear.
Remove Link
If you select part or whole hyperlink area, remove button will be activated.
Pressing remove button will remove the link
Insert Image
Select the area to put image file (jpg, gif, png) and press Image insert/property button.
Image will upload and appear on your document.
Image Property
Select the Image and press the Insert/properties button. A popup window called Image Property will appear.
In this pop up window, you can set image width, height, border.
=> URL (essential input) : specifies the address you want your link to refer to.
=> Tooltip (optional input) : If you move cursor on connected link, tooltip (balloon) will appear
=> Width, Heigth, Border (optional input)
Insert Flash
Select the area to put Flash file(swf) and press Flash insert/property button.
Flash will upload and appear on your document.
Flash Property
Select the Flash and press Flash Insert/property button. A popup window called Flash Property will appear.
In this pop up window, you can set flash width, height, border.
=> URL (essential input) : Flash link address.
=> Width, Height (optional input)
=> Align (optional input) : specifies the alignment of the Flash. You can choose from: Left, Bottom, Middle, Right.
=> Vertical/Horizontal spacing : sets horizontal/vertical spacing.
Insert Media
Select the area to put Movie file (wmv) and press movie insert/property button.
Movie will upload and appear on your document.
(Select the movie and press movie insert/properties then pop up window will appear.)
Media Property
Select the movie and press movie Insert/property button. A popup window called movie Property will appear.
In this pop up window, you can set movie width, height, and player property.
SpecialCharacter
Select the area to put Special-Character and press Special-Character drop down button.
Then, choose the one or more that you would like to insert.
Doc. BackColor
To put document background-color, press document background-color drop down button.
Then, set background color.
Doc. Image
To put document background-image, press document background-image button.
Image will upload and appear on your document.
Doc. Templete
To put document templete, press document templete button. document templete will appear on your document.
